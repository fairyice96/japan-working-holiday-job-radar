這是一個介紹「日本打工度假職缺雷達」計畫的一頁式靜態網站。

## 專案目標

本工具目前正在開發中，目標是整理與日本打工度假相關的公開職缺資訊，協助準備赴日的台灣青年了解工作內容、地點、薪資、日文門檻、住宿、簽證條件與可能風險。

- 預計功能
- 資料來源說明
